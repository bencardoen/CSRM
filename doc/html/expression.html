Subtree crossover operator

static subtreecrossover(left, right, depth=None, rng=None, limitdepth=-1)[source]¶

Perform a subtree crossover in place.

A subtree from left and right are chosen (influenced by seed) and exchanged.

Parameters:
  • left (Tree) – tree to modify with right’s subtree
  • right (Tree) – tree to modify with left’s subtree
  • seed (int) – seed for PRNG (selection of subtree)
  • depth (int) – if not None, forces subtree selection to pick subtrees at the given depth. Else the chosen depth is in [1, min(left.getDepth(), right.getDepth())]
  • limitdepth (int) – if not -1, restricts the depth of the operation.
  • rng (Random) – rng used in calls to select subtrees
class expression.operators.Mutate[source]¶

Bases: object

Mutate a subexpression in the tree

static mutate(tr: expression.tree.Tree, variables, equaldepth=False, rng=None, limitdepth: int = 0, selectiondepth: int = -1)[source]¶

Replace a random node with a new generated subexpression.

If no variables are supplied, the existing set is reused.

This operator is capable of operating using 3 policies specified by its parameters.

With equaldepth set, the resulting mutation will always have the same depth as the original tree. With limitdepth set, the resulting tree will have a depth <= limit With selectiondepth set, the target depth for the mutation point can be specified.

For instance to create a mutation operator that mutates only leafs and replaces them with leafs:
equaldepth=True, limitdepth=0, selectiondepth=tr.getDepth()
Parameters:
  • tr (Tree) – Tree to modify in place
  • variables – set of variables
  • equaldepth (bool) – if set the generated subtree will have the same depth as the node removed, resulting in a mutation which conserves tree depth
  • limitdepth (int) – if not 0 prevent the mutation from growing a resulting tree with depth larger than limit
  • selectiondepth (int) – if not -1 specify at which depth the insertion point is chosen
  • rng (Random) – prng used to generate the new subtree and its attaching location
